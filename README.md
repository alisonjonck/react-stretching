## Install

    $ git clone https://github.com/alisonjonck/react-stretching
    $ cd react-stretching
    $ npm install

## Run tests

    $ npm test

## Run tests with debug

    $ npm run test-debug
On Chrome open chrome://inspect

## Start & watch

    $ npm start

## Build for production

    $ npm run build

## Run lint

    $ npm run lint
